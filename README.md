## Development server


`cd CIS3110-Project-1`.
Then, `npm install`.
Run `json-server -w db.json`.
Finally, on another terminal, make sure you're in the CIS3110-Project-1 directory then run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
